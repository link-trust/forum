# Link Trust Forum

基于 Astro 开发的社区论坛

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 项目结构

```
link-trust-forum/
├── src/
│   ├── layouts/     # 页面布局
│   └── pages/       # 路由页面
├── public/          # 静态资源
├── astro.config.mjs # Astro 配置
└── package.json     # 项目依赖
```
